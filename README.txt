[ iframe 없는 웹 사이트 뷰어 프로젝트 구동 방법 ]

HTML 단독으로는 브라우저의 CORS 및 X-Frame-Options 보안 정책 때문에 다른 사이트를 읽어올 수 없습니다.
본 프로젝트는 Node.js 서버를 프록시(Proxy) 삼아 상대방 사이트의 HTML 소스를 긁어온 뒤 화면에 뿌려주는 풀스택 예제 폴더입니다.

■ 실행 순서

1. Node.js 설치 확인
   - 컴퓨터에 Node.js가 설치되어 있어야 합니다. (없다면 https://nodejs.org 에서 LTS 버전 설치)

2. 터미널(CMD 등) 열기 및 폴더 이동
   - 본 압축을 푼 폴더(site_viewer_project) 내부로 터미널 경로를 이동합니다.
   - 예: cd site_viewer_project

3. 필요한 모듈 설치 (express, axios)
   - 터미널에 아래 명령어를 입력하여 모듈을 설치합니다.
   > npm install

4. 서버 구동
   - 설치가 끝나면 아래 명령어로 서버를 켭니다.
   > npm start

5. 브라우저 접속
   - 크롬 등 웹 브라우저를 열고 주소창에 아래 주소를 입력하세요.
   - http://localhost:3000

■ 참고 사항
- 백엔드단에서 상대경로(/assets/...)를 절대경로로 바꾸는 기초적인 정규식 처리가 들어가 있어, 이전 단판 코드보다 스타일이 덜 깨집니다.
- 다만, 프론트엔드가 리액트/뷰 등 단일 페이지 애플리케이션(SPA) 빌드본이거나 자체 쿠키/보안 토큰 인증 메커니즘이 엄격한 사이트(네이버 로그인창 등)는 스크래핑 방식으로 가져왔을 때 내부 스크립트 에러가 날 수 있습니다. 이는 프론트엔드 레벨 스크래핑의 기술적 한계점인 점 참고 부탁드립니다!
